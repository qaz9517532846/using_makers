# using_makers
ROS maker Example.
Detail about imform [this](http://wiki.ros.org/rviz/DisplayTypes/Marker).

show basic shape in rviz
``` bash
$ rosrun using makers basic_shapes
```
``` bash
$ rosrun rviz rviz
```

show points and lines in rviz
``` bash
$ rosrun using makers points_and_lines
```
``` bash
$ rosrun rviz rviz
```
